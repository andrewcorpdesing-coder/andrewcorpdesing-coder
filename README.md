<div align="center"><h3>🧠 @animakit — cognitive architecture for LLM agents</h3></div>

Open-source packages **extracted from a production AI agent** (~53 sprints of real traffic). Every README number is reproducible from a script.

- **[@animakit/complexity-scorer](https://www.npmjs.com/package/@animakit/complexity-scorer)** — classify LLM tasks in <1ms, 0 tokens → **31.6% cost reduction** (honestly measured). *In production.*
- **[@animakit/homeostasis](https://www.npmjs.com/package/@animakit/homeostasis)** — stress/dopamine state for agents (Yerkes-Dodson, self-tuning bandit). Verified benchmarks; **running in production via dogfooding.**
